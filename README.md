# 戏曲唱词

### 字幕版

> ### 介绍
>
> 主要存放黄梅戏，越剧等戏曲唱词．详情请查看[戏曲唱词信息](https://xlgp.gitee.io/opera-lyrics-doc/#/).

> ### LRC 文件格式

```
 [ti:唱段名]
 [oti:唱段名]
 [jz:剧种]
 [jm:剧目]
 [ojm:剧目]
 [offset:-1]
 [bz:备注] //可选
 [00:01.02]唱词１
 [00:20.34]<结束>
```
