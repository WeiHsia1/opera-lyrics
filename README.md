# 戏曲唱词

> ### 介绍
主要存放黄梅戏，越剧等戏曲唱词．

> ### LRC文件格式
```
 [ti:标题]
 [jz:剧种]
 [jm:剧目]
 [offset:-1]
 [bz:备注] //可选
 [00:01.02]唱词１
```

> ### LRC demo
```
[ti:一月思念]
[jz:黄梅戏]
[jm:龙女]
[offset:-1]
[00:09.69]一月思念如痴如醉
[00:14.82]相爷带的佳音归
[00:22.83]他才也高貌也美
```
